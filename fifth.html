  • File Manager : nnn
  • Text Editor : vim
  • Music Player : moc
  • Pictures : qiv
  • Web browser : Chromium
  • Plotting : gnulplot
  • Schematic drawing : xcircuit
  • Task Manger : htop
  • System Notification : herbe
  • Pdf reader : xreader
